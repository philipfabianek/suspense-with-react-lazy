## Suspense with React.lazy()

This is an example of how you can use Suspense with React.lazy().

### Installation

1. Clone the project
   ```sh
   git clone https://github.com/philipfabianek/suspense-with-react-lazy.git
   ```
2. Install NPM packages
* npm
   ```sh
   npm install
   ```
* yarn
   ```sh
   yarn
   ```
3. Start project
   ```sh
   npm start
   ```
